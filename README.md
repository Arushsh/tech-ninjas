# Cloth Showroom Website

This project is a simple and responsive frontend website for a cloth showroom, developed using HTML, CSS, and JavaScript.  
It allows users to browse different categories of clothing such as men’s wear, women’s wear, and kids’ wear with a clean and modern layout.

---

## Features

- Responsive design for all screen sizes  
- Separate sections for Men, Women, and Kids  
- Add-to-cart interface (frontend only)  
- Search functionality for finding products  
- Smooth hover and transition animations  
- Contact form for feedback or inquiries  

---

## Technologies Used

- HTML5 – Structure and layout  
- CSSa3 – Styling and animations  
- JavaScript – Dynamic elements and interactivity
